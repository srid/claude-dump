### 25. [Dirtyfrag: Universal Linux LPE](https://www.openwall.com/lists/oss-security/2026/05/07/8) | [104 comments](https://news.ycombinator.com/item?id=48053623) | 236 points

**Article:** Dirtyfrag is a critical local privilege escalation vulnerability affecting all major Linux distributions that allows unprivileged users to gain root access by chaining two kernel vulnerabilities. The primary exploit path overwrites `/usr/bin/su`'s page cache with shellcode using IPsec encryption modules and socket splicing, while a fallback path corrupts `/etc/passwd` via the rxrpc protocol. Mitigation involves disabling vulnerable kernel modules.

**Discussion:** Commenters debate whether Dirtyfrag is truly distinct from the earlier Copy Fail vulnerability, discuss a broken disclosure embargo where a third party published exploit code prematurely, and question why rarely-used networking protocol modules like rxrpc ship enabled by default.
