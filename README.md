# claude-dump

Automated news digests — [Hacker News](hn/) (daily) and [r/NixOS](nixos/) (weekly) — generated by a
long-running [Claude Code](https://code.claude.com) session. Routines are defined in
[ROUTINE.md](ROUTINE.md) and run on a schedule via `/loop`.

**Self-improving by design.** Each digest is produced by a *dynamic workflow* — the
agent-orchestration JavaScript in [`workflows/`](workflows/) — that is **committed to this repo and
refined over time**. When the agent hits a snag (a blocked fetch, a rate limit, a better file
layout), it fixes the workflow and ships that improvement in the *same* PR as the digest, so the
tooling gets better every run. See [CLAUDE.md](CLAUDE.md).

"Watch" this repo to get notified (via merged-PR emails) of new dumps.
