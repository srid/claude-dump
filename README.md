# claude-dump

Dumping ground for automated digests (e.g. Hacker News top stories) generated by a long-running
[Claude Code](https://code.claude.com) session.

Tasks are defined in [ROUTINE.md](ROUTINE.md) and run on a schedule via `/loop` — see
[CLAUDE.md](CLAUDE.md).

"Watch" this repo to get notified (via merged-PR emails) of new dumps.
